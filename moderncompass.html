Web/Mobile Application - July 2017

A new upcoming app that will make outdoor exploring safer.

What is Modern Compass?

Modern Compass is a cross-platform outdoor activity companion app for outdoor users of all levels. The mobile app features assigning an emergency contact with the abilities to track the users GPS location and footprints.

Workflow

The project is being developed by a four man team. This project utilizes crowdsourcing to support the various amount of outdoor activities for users.
