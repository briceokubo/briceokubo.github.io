TecHub Website

Overview

I created wireframes, mockups and prototype for the newly branded department web page in University of Hawai'i Manoa Bookstore. Through the designing process I made some UX design based decision.

Desiging Phase

Research

The previous department web page was located on its own domain and separate from the main University of Hawai'i Bookstore e-commerce website. The objected of the research was to create web page compareable to Manoa Bookstore e-commerce and incorporate a more user friendly design.

Wireframes

The director wanted a simple navigational website to help students and faculty/staff be able to use the website with ease.

The previous website had all the tabs located on the navbar. The new design utilizes quick banner links on the main page, so customers will be able to easily access popular section of the website.

User Flow

With the power of Adobe Experience Design it's easy to display and demostrate the user flow of wireframes to the director and marketing team.

Problems/ Solutions

Within the TecHub department, they have limited access to update or change the website. The marketing department is the only one with all access to the code. With the constant change of products, the marketing department may not be able to keep up to date. I created a solution to use the previous department CMS web page to create PDFs pages to display non-e-commerce products for the new web page.

Conclusion

This project was a great experience for myself. From my experience in UX/UI design and IT support of the department, I was able to translate my knowledge and experience with customers to the UX design of the website to target the right audience.
